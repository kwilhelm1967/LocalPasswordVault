                         STEP 1: INSTALLATION
================================================================================

Windows:
--------
1. Double-click LocalPasswordVault-Setup.exe
2. Follow the installation wizard
3. Click "Install" and wait for completion
4. Launch from desktop shortcut or Start Menu

macOS:
------
1. Open LocalPasswordVault.dmg
2. Drag Local Password Vault to Applications folder
3. Open from Applications
4. If prompted, click "Open" to confirm

Linux:
------
1. Make executable: chmod +x LocalPasswordVault.AppImage
2. Run: ./LocalPasswordVault.AppImage
