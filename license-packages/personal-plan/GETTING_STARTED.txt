================================================================================
                    LOCAL PASSWORD VAULT - GETTING STARTED
                              Personal Plan Guide
================================================================================

Welcome! This guide will walk you through setting up your secure password vault.

================================================================================
                         STEP 1: INSTALLATION
================================================================================

Windows:
--------
1. Double-click LocalPasswordVault-Setup.exe
2. Follow the installation wizard
3. Choose your installation directory (default recommended)
4. Click "Install" and wait for completion
5. Launch from desktop shortcut or Start Menu

macOS:
------
1. Open LocalPasswordVault.dmg
2. Drag Local Password Vault to Applications folder
3. Open from Applications
4. If prompted, click "Open" to confirm (first launch only)

Linux:
------
1. Make the AppImage executable: chmod +x LocalPasswordVault.AppImage
2. Run: ./LocalPasswordVault.AppImage
   OR install the .deb package: sudo dpkg -i LocalPasswordVault.deb

================================================================================
                         STEP 2: LICENSE ACTIVATION
================================================================================

1. Launch Local Password Vault
2. You'll see the license activation screen
3. Enter your license key (from your purchase email)
4. Click "Activate License"
5. Your license is now active on this device

License Key Format: PERS-XXXX-XXXX-XXXX

Keep your license key safe! You'll need it if you reinstall.

================================================================================
                         STEP 3: CREATE MASTER PASSWORD
================================================================================

Your master password is the ONLY way to access your vault.

Requirements:
- Minimum 12 characters (longer is better)
- Mix of uppercase, lowercase, numbers, symbols
- Something memorable to YOU but hard for others to guess

⚠️  CRITICAL: Your master password CANNOT be recovered!
    We don't store it anywhere. If you forget it, your data is lost forever.

Tips:
- Use a passphrase: "My dog Bella loves 3 walks daily!"
- Write it down and store in a physical safe location
- Don't use the same password you use elsewhere

================================================================================
                         STEP 4: ADD YOUR FIRST PASSWORD
================================================================================

1. Click "+ Add Account" button
2. Fill in the details:
   - Account Name (e.g., "Gmail", "Netflix")
   - Username/Email
   - Password (or click generate for a strong one)
   - Website URL (optional)
   - Category (Banking, Shopping, etc.)
3. Click "Save"

Your password is now encrypted and stored locally!

================================================================================
                         STEP 5: USING THE FLOATING PANEL
================================================================================

The Floating Mini Vault gives quick access to your passwords:

1. Click the minimize button in the main vault
2. A small floating panel appears
3. Search for accounts quickly
4. Click to copy passwords
5. Click the expand button to return to full vault

Great for filling in login forms while browsing!

================================================================================
                         STEP 6: PASSWORD GENERATOR
================================================================================

Never create weak passwords again:

1. When adding/editing an account, click the refresh icon
2. Adjust settings:
   - Length (16+ recommended)
   - Include uppercase, lowercase, numbers, symbols
3. Click "Generate"
4. The strength meter shows how secure it is

================================================================================
                         STEP 7: IMPORT EXISTING PASSWORDS
================================================================================

Import from other password managers:

1. Go to Settings > Import
2. Choose your format (CSV, JSON, or encrypted backup)
3. Select your export file
4. Review imported entries
5. Click "Import"

Supported imports:
- Chrome/Firefox exported CSV
- LastPass export
- 1Password export
- Bitwarden export
- Generic CSV

================================================================================
                         STEP 8: CREATE A BACKUP
================================================================================

Protect against data loss:

1. Go to Settings > Export
2. Choose "Encrypted Backup"
3. Enter a backup password (can be different from master password)
4. Save the backup file
5. Store it safely (USB drive, external hard drive, etc.)

Recommendation: Create weekly backups!

================================================================================
                         STEP 9: ENABLE AUTO-LOCK
================================================================================

Automatically lock your vault when you step away:

1. Go to Settings
2. Find "Auto-Lock Timeout"
3. Set your preferred time (5 minutes recommended)
4. The vault will lock after inactivity

================================================================================
                              KEYBOARD SHORTCUTS
================================================================================

Ctrl/Cmd + N     - Add new account
Ctrl/Cmd + F     - Search
Ctrl/Cmd + L     - Lock vault
Escape           - Close dialogs
