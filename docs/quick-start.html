Get started with Local Password Vault in 5 minutes

1Install

  1. Download from LocalPasswordVault.com
  2. Run the installer
  3. Launch the application

2Create Your Vault

  1. Enter a master password
    • Minimum 12 characters
    • Mix of letters, numbers, and symbols
    You cannot recover this password
  2. Save your recovery phrase
    • Write down the 12 words
    • Store it securely
    • This is your backup
  3. Click "Create Vault"

3Activate License

Option A: Free Trial

  1. Click "Start Free Trial"
  2. Enter your email
  3. Receive trial key via email
  4. Enter trial key
  5. 7 days free to try all features

Option B: Purchase License

  1. Click "Purchase License"
  2. Choose plan (Personal $49 or Family $79)
  3. Complete purchase
  4. Receive license key via email
  5. Enter license key
  6. Lifetime access - no subscriptions

4Add Your First Password

  1. Click the "+" button
  2. Fill in:
    • Title: "Gmail"
    • Username: your email
    • Password: click key icon to generate
    • URL: gmail.com
    • Category: Email
  3. Click "Save"
Done! Your password is now encrypted and stored securely.

5Use Your Passwords

View Password

  1. Click an entry
  2. Click eye icon to reveal password
  3. Click copy icon to copy password

Generate Strong Password

  1. Click key icon in password field
  2. Set length (16+ recommended)
  3. Enable all options
  4. Click "Generate"
  5. Click "Copy"

Search

  • Type in the search bar
  • Filter by category
  • Press Ctrl+F / Cmd+F

Essential Settings

Auto-Lock (Recommended)

  1. Go to Settings
  2. Set Auto-Lock to 5 minutes
  3. Vault locks automatically when idle

Clipboard Clear (Recommended)

  1. Go to Settings
  2. Set Clipboard Clear to 30 seconds
  3. Passwords clear from clipboard automatically

Keyboard Shortcuts

Action Windows/Linux Mac
New Entry Ctrl+N Cmd+N
Search Ctrl+F Cmd+F
Settings Ctrl+, Cmd+,
Copy Password Ctrl+C Cmd+C
